# Experiment 5 – Toggle Button

&#x20;

#### **Aim**



To develop an Android application that uses a ToggleButton and displays a message using Toast when the button is switched ON or OFF.



#### **Algorithm**



1. Start the application.
2. Create a new Android project.
3. Design the layout using RelativeLayout.
4. Add a ToggleButton in the XML file.
5. Set properties like:
* textOn = "ON"
* textOff = "OFF"

6\. Open MainActivity.java.

7\. Initialize the ToggleButton using findViewById().

8\. Set a listener using setOnCheckedChangeListener().

9\. Check the state of the ToggleButton:

* If ON → Display Toast message “Toggle is ON”.
* If OFF → Display Toast message “Toggle is OFF”.

10\. Run the application.

11\. Verify the output.
