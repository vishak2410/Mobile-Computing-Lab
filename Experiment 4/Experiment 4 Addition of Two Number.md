###### Experiment 4: Addition of Two Numbers



###### Aim



* To develop an Android application that performs addition of two numbers entered by the user. 



###### Algorithm



* Start Android Studio and create a new Android project.



* Design the layout using:



* Two EditText fields for input numbers.



* One Button for addition.



* One Button for reset.



* One TextView to display the result.



* Open MainActivity.java.



* Link all UI components using findViewById().



* Set an OnClickListener for the Add button.



* Read numbers from the EditText fields.



* Convert the input values into numeric format.



* Add the two numbers.



* Display the result in the TextView.



* Set another OnClickListener for the Reset button.



* Clear the input fields and result.



* Run the application and test the addition.
