###### Experiment 2: Graphic Primitives Using Canvas



###### Aim



* To create an Android application that demonstrates graphic primitives such as line, rectangle, circle, oval, and text using Canvas. 



###### Algorithm



* Start Android Studio and create a new Android project.



* Create a custom class extending View.



* Declare a Paint object for drawing graphics.



* Override the onDraw() method.



-->Use Canvas methods to draw shapes:



&nbsp;	drawLine() for line



&nbsp;	drawRect() for rectangle



&nbsp;	drawCircle() for circle



&nbsp;	drawOval() for oval



&nbsp;	drawText() for text



* Set colors and styles using the Paint object.



* In MainActivity, create an object of the custom class.



* Use setContentView() to display the custom view.



* Run the application to view the graphic shapes on the screen.
