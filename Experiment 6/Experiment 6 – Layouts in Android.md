# Experiment 6 – Layouts in Android



(Linear, Relative, Table Layout)



#### Aim



To create an Android application that demonstrates different types of layouts such as LinearLayout, RelativeLayout, and TableLayout and navigate between them using buttons.



#### Algorithm



1. Start the application.
2. Create a new Android project.
3. Design the main layout using LinearLayout.
4. Add three buttons:
* Linear Layout
* Relative Layout
* Table Layout

5\. Create three separate activities:

* LinearActivity
* RelativeActivity
* TableActivity

6\. Set click listeners for each button:

* On click → Open respective activity using Intent.

7\. Design each layout:

* LinearLayout: Arrange components vertically.
* RelativeLayout: Position elements relative to each other.
* TableLayout: Arrange data in rows and columns.

8\. Add all activities in AndroidManifest.xml.

9\. Run the application.

10\. Click buttons and verify navigation to different layouts.

11\. Observe how each layout behaves.
